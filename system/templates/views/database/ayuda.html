Para manejar todo lo relacionado con las bases de datos de rasgos y objetos:

Se accede a la página "Bases de datos".


Para crear una nueva base de datos de rasgos:

Se selecciona el botón "Nueva" para crear una nueva base de datos de rasgos.

Se escribe el nombre de la base de datos en el campo "Nueva base de datos".

Se escribe el nombre del rasgo en el campo "Nombre del rasgo".

Se adicionan todos los rasgos que se deseen en el botón "Adicionar".

Se selecciona el criterio de comparación.

En caso de seleccionar el criterio "Nominal", se adicionan tantos dominios como se deseen crear con el botón Adicionar y se escriben los valores de los dominios, solo admite letras y caracteres en blanco.

En caso de seleccionar el criterio "Booleano", no se deberán escribir valores para el dominio ya que por defecto serán "Si" y "No"

En caso de seleccionar el criterio "Cuantitativo", se adicionan tantos dominios como se deseen crear con el botón Adicionar y se escriben los valores de los dominios, solo admite valores numéricos.

En caso de seleccionar el criterio "Por intervalos", se adicionan tantos intervalos como se deseen crear con el botón Adicionar, solo admite valores numéricos, donde el valor del campo Inicio debe ser menor que el del valor del campo Fin, y donde los valores de los campos Fin deben ser menores que los campos Inicio del intervalo anterior.

Se selecciona el botón "Finalizar".


Para modificar una base de datos de rasgos:

Se busca en la "Lista de rasgos" y se selecciona el botón "Modificar" de la base de datos que se desee modificar.

Se procede a modificar los valores que se desee.

Se selecciona el botón "Finalizar".


Para eliminar una base de datos de rasgos:

Se busca en la "Lista de rasgos" y se selecciona el botón "Eliminar" de la base de datos que se desee eliminar.

Selecciona el botón "Eliminar".


Para adicionar objetos a una base de datos de rasgos:

Se busca en la Lista de objetos y se selecciona el botón "Adicionar" de la base de datos que se desee añadir objetos.

Se adicionan todos los objetos que se deseen en el botón "Adicionar".

Se escribe el nombre del objeto en el campo "Nombre del objeto".

Se seleccionan los dominios a los objetos que se deseen para cada rasgo.

Se selecciona el botón "Finalizar".


Para modificar objetos en una base de datos de rasgos:

Se busca en la Lista de objetos y se selecciona el botón "Modificar" de la base de datos que se desee modificar objetos.

Se modifican los objetos y los dominios de los rasgos que se deseen.

Se selecciona el botón "Finalizar".

Para manejar todo lo relacionado con las bases de datos de rasgos y objetos:

Se accede a la página "Clasificación".

Para crear los grupos:

Se selecciona el botón "Clasificar" en la base de datos deseada.

Se selecciona el tipo de umbral de semejanza.

En caso de seleccionar "Criterio de expertos", se escribe un número entre 0 y 1 en el campo de texto.

En caso de seleccionar "Calculado por el sistema", se selecciona el tipo de criterio de experto: "Mínimo", "Máximo" o "Media".

Se selecciona el botón "Finalizar".


Para ver los grupos creados:

Se selecciona el botón "Ver grupos" en la base de datos donde se crearon los grupos.
